## 单例模式 Singleton Pattern
- 指一个类只有一个实例，且该类能自行创建这个实例的一种模式

## 特点
### 单例模式有 3 个特点：
- 单例类只有一个实例对象；
- 该单例对象必须由单例类自行创建；
- 单例类对外提供一个访问该单例的全局访问点。
